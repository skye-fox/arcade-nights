# arcade-nights
Colorful neovim dark-mode theme

## Installation
```lua
return {
	"skye-fox/arcade-nights",
	priority = 1000,
	opts = {},
}
```

### Configuration
```lua
opts = {
	plugins = {
		gitsigns = (true/false),
		mini_icons = (true/false),
		render_markdown = (true/false),
		snacks = (true/false),
	},
}
```
